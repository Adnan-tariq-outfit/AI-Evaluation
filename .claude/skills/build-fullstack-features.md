AGENT SKILL: FULLSTACK FEATURE DEVELOPMENT

Name: build_fullstack_feature

Description:
Skill used to implement a complete feature across frontend and backend in a structured way.

Inputs:

* feature_name
* description
* api_requirements
* database_schema
* ui_requirements

Execution Steps:

1. Analyze Requirements

   * Understand feature goal
   * Identify API endpoints
   * Identify UI components
   * Define database schema

2. Backend Implementation

   * Create module
   * Define DTOs
   * Create controller endpoints
   * Implement service logic
   * Implement repository/database layer
   * Add validation and error handling

3. API Contract

   * Define request/response structure
   * Ensure consistent API format
   * Add pagination or filtering if required

4. Frontend Implementation

   * Create feature folder
   * Build reusable components
   * Implement API service
   * Add state management
   * Handle loading/error states

5. Integration

   * Connect frontend with backend APIs
   * Test data flow
   * Ensure type safety

6. Optimization

   * Reduce unnecessary API calls
   * Optimize rendering
   * Apply caching where needed

Output:

* Backend file structure
* Backend code
* Frontend file structure
* Frontend code
* API contract
* Short explanation

Rules:

* Follow modular architecture
* Maintain clean separation of concerns
* Use TypeScript types everywhere
* Ensure production-ready code
